Требования приложения:
- PHP 7.1 или выше;
- MySQL 5.6 или выше;
- Apache 2.4.34 или выше;

Для запуска приложения необходимо:
1. Отредактировать файл /config/db_config.php в соответствии с настройками своей БД (вписать имя хоста, БД, имя пользователя, пароль).
2. Из браузера запустить скрипт <ваш_тестовый_домен>/install.php.
3. Перейти по адресу вашего тестового домена и пользоваться приложением.
4. Enjoy!
